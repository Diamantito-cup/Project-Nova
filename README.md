> *Un juego que nació por pura curiosidad*

---

Project-nova es un juego Metroidvania en desarrollo por mí y mi equipo, son libres de usar parte de este codigo como base para iniciar,
pero, está prohibida su distribución sin consentimiento de nosotros los desarrolladores :3

---

<h2 align="center">Datos interesantes sobre el desarrollo del juego</h2>

Actualmente el juego `Project-Nova` está en una pausa de desarrollo por temas personales (a quien le delegué el arte visual se anda tardando en entregar los sprites >:V), pero, para compensarlo, les vamos a decir el como nacio el proyecto, yo, **[Ange^_^](https://github.com/Diamantito-cup)** estaba cacharreando con Unity y jugando un poco con claude a ver que pasaba, luego, vi la oportunidad, la aproveché y empecé el desarrollo del juego, todo avanzó bastante rapido la vdd, no hubo muchos problemas, pero el desarrollo paró cuando, pues, llegamos al limite de lo que el código puede hacer, así que tocóhacer la clasica de esperar el arte :D.

Pero por el resto, todo muy bien, el que quiera ayudar pues, bienvenido :P
