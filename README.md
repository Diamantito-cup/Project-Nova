## Introducción a este proyecto

Project-nova (nombre provicional) es un juego Metroidvania en desarrollo por mí y mi equipo, son libres de usar parte de este codigo como base para iniciar,
pero, está prohibida su distribución sin consentimiento de nosotros los desarrolladores, sin mas que decir, que disfruten leyendo :3
